### 1. Spring4特性

| Spring版本 | 特性                                                         |
| ---------- | ------------------------------------------------------------ |
| Spring1.x  | XML配置                                                      |
| Spring2.x  | 注解配置                                                     |
| Spring3.x  | 从Spring3.x开始提供Java类配置方式，可以完全替代XML配置<br />而且Spring4.x和SpringBoot都推荐使用Java类配置方式 |
